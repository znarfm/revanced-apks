# ReVanced Magisk Module

[![CI](https://github.com/znarfm/revanced-apks/actions/workflows/ci.yml/badge.svg?event=schedule)](https://github.com/znarfm/revanced-apks/actions/workflows/ci.yml) [![GitHub tag](https://img.shields.io/github/tag/znarfm/revanced-apks?include_prereleases=&sort=semver&color=purple)](https://github.com/znarfm/revanced-apks/releases/) [![License](https://img.shields.io/badge/License-MIT-purple)](#license)

An extensive ReVanced builder that creates up-to-date Magisk modules and non-root APKs from

**Note:** If you're using Magisk modules, use [zygisk-detach](https://github.com/j-hc/zygisk-detach) to detach YouTube and YT Music from the Play Store to avoid conflicts.

-----

## Key Features

<details>
    <summary>View All Features</summary>

* **Comprehensive Support:** Supports all current and future ReVanced and ReVanced Extended apps.
* **Build Flexibility:** Can build both Magisk modules and non-root APKs.
* **Always Up-to-Date:** Daily updates with the latest app and patch versions.
* **Optimized:** APKs and modules are optimized for size.
* **Magisk Module Specifics:**
  * Recompiles invalidated odex files for faster performance.
  * Receives updates directly from the Magisk app.
  * Does not trigger root detections or break SafetyNet.
  * Handles the correct installation of the stock app.
  * Supports both Magisk and KernelSU.

</details>

### Customizing Your Build

Want to include or exclude specific patches or patch other apps? Follow these steps to use this repo as a template.

1. **Star the repo** :eyes: and then use it as a [template](https://github.com/new?template_name=revanced-magisk-module&template_owner=j-hc).
2. **Customize your configuration** by editing `config.toml`. You can use the `rvmm-config-gen` online tool for a user-friendly interface: [rvmm-config-gen](https://j-hc.github.io/rvmm-config-gen/). See `CONFIG.md` for more details on configuration options.
3. **Run the build workflow** by manually triggering the build [suspicious link removed].
4. **Grab your builds** from the [suspicious link removed] page once the workflow is complete.

### Building Locally

You can also build the modules and APKs directly on your device.

#### On Termux

```sh
bash <(curl -sSf https://raw.githubusercontent.com/j-hc/revanced-magisk-module/main/build-termux.sh)
```

#### On Desktop

```sh
git clone https://github.com/j-hc/revanced-magisk-module
cd revanced-magisk-module
./build.sh
```
